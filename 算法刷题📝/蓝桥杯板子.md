### DFS
例题：整数拆分，把大小为$n$的一个整数分为$m$个整数。 如 $6 = 1 + 2 + 3$
- 考虑用for循环来做，需要开m重循环
```
// for example 6 = 1 + 2 + 3
for(int i = 1; i <= n; i ++) {
	for(int j = i; j <= n; j ++) {
		for(int k = j; k <= n; k ++){
			if(i + k + j == n)
				printf("%d = %d + %d + ", n, i, j, k )
		}
	}
}
```
