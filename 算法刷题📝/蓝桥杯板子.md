### DFS
例题：整数拆分，把大小为$n$的一个整数分为$m$个整数。 如 $6 = 1 + 2 + 3$
- 考虑用for循环来做，需要开m重循环
```
// for example 6 = 1 + 2 + 3
for(int i = 1; i <= n; i ++) {
	for(int j = i; j <= n; j ++) {
		for(int k = j; k <= n; k ++){
			if(i + k + j == n)
				printf("%d = %d + %d + %d\n", n, i, j, k )
		}
	}
}
```

- 考虑使用递归来做
```
int m, arr[103];

void dfs(int n, int i, int a) {
	if(n == 0) {
		for(int j = 1; j <= i - 1; j ++) printf("%d", arr[j])
	}
	if(j <= m) {
		for(int j = a; j <= n; j ++) {
			arr[i] = j;
			dfs(n - j, i + 1, j);
		}
	}
}
```
